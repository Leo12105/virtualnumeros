# NumVirtual

Página web de números virtuales temporales de diferentes países.

### Características
- Números de USA, UK, Canadá, México, Brasil, España, India y más
- Diseño moderno y oscuro
- Fácil de actualizar
- Totalmente gratis y open source

### Cómo usar
1. Solo abre el `index.html`
2. Copia el número que necesites

### Cómo actualizar los números
Edita el archivo `js/numbers.js` y agrega más números.
